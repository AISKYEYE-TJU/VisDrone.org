## 📋 常用检查

### 检查数据完整性
```
请检查VisDrone网站数据文件的完整性：
1. 检查所有TS文件是否有语法错误
2. 检查ID是否有重复
3. 检查必填字段是否完整
4. 检查图片链接是否有效
```

### 排序数据
```
请将VisDrone网站的数据按以下规则排序：
- 模型：按Stars降序
- 数据集：按Stars降序
- 论文：按年份降序
- 团队：按角色和加入时间排序
```

## 使用说明

1. **复制模板**：选择合适的模板，复制到对话框
2. **填写内容**：将方括号 [] 内的内容替换为实际信息
3. **发送请求**：将完整的Prompt发送给AI助手
4. **验证结果**：检查更新后的文件是否正确

## 注意事项

- ID必须唯一，不能与现有数据重复
- 日期格式：YYYY-MM 或 YYYY-MM-DD
- 图片推荐使用AI生成的URL
- 提交前请验证TypeScript语法正确
