# VisDrone Admin 管理后台使用指南

## 一、登录管理

### 1.1 访问地址

```
https://aiskyeye.com/visdrone/admin
```

### 1.2 默认账户

| 角色 | 邮箱 | 密码 |
|------|------|------|
| 管理员 | admin@visdrone.org | visdrone |

### 1.3 登录流程

1. 访问管理后台地址
2. 输入邮箱和密码
3. 点击"登录"按钮
4. 登录成功后自动跳转到管理面板

## 三、新闻管理

### 3.1 添加新闻

1. 点击"新闻管理"
2. 点击"添加新闻"按钮
3. 填写表单：
   - **标题**：新闻标题
   - **链接**：原文 URL
   - **日期**：YYYY-MM 格式（如 2026-03）
   - **摘要**：简短描述
   - **分类**：选择分类（achievement|academic|activity|other）
   - **图片**：上传或输入 URL
4. 点击"保存"

### 3.2 编辑新闻

1. 在新闻列表中找到要编辑的新闻
2. 点击操作列的"编辑"按钮
3. 修改表单内容
4. 点击"保存"

### 3.3 删除新闻

1. 在新闻列表中找到要删除的新闻
2. 点击操作列的"删除"按钮
3. 确认删除

## 四、学术活动管理

### 4.1 学术活动类型

| 类型 | 值 | 说明 |
|------|-----|------|
| 组会 | group_meeting | 实验室组会报告 |
| 特邀报告 | invited_talk | 邀请的学术报告 |
| 学术会议 | workshop | 参加的学术会议 |

### 4.2 学术活动分组

| 分组 | 值 | 说明 |
|------|-----|------|
| 学习范式 | learning | 学习范式研究组 |
| 多模态学习 | multimodal | 多模态学习研究组 |
| 具身智能 | embodied | 具身智能研究组 |

### 4.3 添加组会

1. 点击"学术活动"
2. 点击"添加组会"按钮
3. 填写表单：
   - **标题**：论文/报告标题
   - **日期**：YYYY-MM-DD 格式
   - **分组**：选择分组（learning|multimodal|embodied）
   - **PPT链接**：可选
   - **论文链接**：可选
4. 点击"保存"

### 4.4 添加特邀报告

1. 点击"学术活动"
2. 点击"添加特邀报告"按钮
3. 填写表单：
   - **标题**：报告标题
   - **日期**：YYYY 或 YYYY-MM-DD
   - **PPT链接**：可选
4. 点击"保存"

## 五、数据同步

### 5.1 同步到数据库

将本地 TypeScript 文件数据同步到 Supabase 数据库：

1. 点击"数据同步"
2. 查看各模块的本地 TS 数量和数据库数量
3. 点击需要同步的模块的"同步到数据库"按钮
4. 等待同步完成

**注意**：
- 首次同步学术活动前，需在 Supabase 执行 SQL 添加 `group` 列
- 如果同步失败，检查 Supabase 表的 RLS 策略

### 5.2 从数据库导入

将 Supabase 数据库数据导入到本地 TypeScript 文件：

1. 点击"数据同步"
2. 点击"从数据库导入"按钮
3. 确认导入

## 六、常见问题

### 6.1 同步失败

**原因**：Supabase 表启用了 RLS 策略

**解决方案**：在 Supabase SQL Editor 中执行：
```sql
ALTER TABLE visdrone_seminars DISABLE ROW LEVEL SECURITY;
```

### 6.2 学术活动显示为 0

**原因**：数据库 `group` 列不存在

**解决方案**：在 Supabase SQL Editor 中执行：
```sql
ALTER TABLE visdrone_seminars ADD COLUMN "group" TEXT DEFAULT 'learning';
```

### 6.3 登录失败

**检查项**：
1. 邮箱密码是否正确
2. 浏览器是否允许 cookies
3. 网络连接是否正常

---

## 七、安全注意事项

1. **账户安全**：不要在公开场合泄露管理员密码
2. **操作谨慎**：删除操作不可恢复，请谨慎操作
3. **定期备份**：重要数据操作前建议先备份
4. **权限控制**：仅管理员账户可访问管理后台
